# 小红书 (Xiaohongshu / RedNote)

## Video Specs

| Parameter | Value |
|-----------|-------|
| Aspect Ratio | 9:16 (竖屏) |
| Resolution | 720x1280 |
| Duration | 15-60s |
| Format | MP4 |
| Max File Size | 100MB |

## Content Tips

- 竖屏优先，9:16 是最佳比例
- 前 3 秒抓住注意力
- 15-30s 短视频效果最好
- 字幕/文字叠加提升完播率
- 封面图很重要，建议单独生成

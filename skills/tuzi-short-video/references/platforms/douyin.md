# 抖音 (Douyin / TikTok)

## Video Specs

| Parameter | Value |
|-----------|-------|
| Aspect Ratio | 9:16 (竖屏) |
| Resolution | 1080x1920 |
| Duration | 15-60s |
| Format | MP4 |
| Max File Size | 128MB |

## Content Tips

- 竖屏 9:16 是标准格式
- 支持更高分辨率（1080p）
- 前 1-3 秒决定用户是否继续观看
- 15s 短视频适合快节奏内容
- 60s 适合教程/故事类内容

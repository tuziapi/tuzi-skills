# X / Twitter

## Video Specs

| Parameter | Value |
|-----------|-------|
| Aspect Ratio | 16:9 (横屏) |
| Resolution | 1280x720 |
| Duration | 5-140s |
| Format | MP4 |
| Max File Size | 512MB |

## Content Tips

- 横屏 16:9 是主流格式
- 短视频（15-30s）传播效果好
- 支持较长视频（最长 140s）
- 自动播放无声，前几秒需视觉吸引力
- 建议添加字幕

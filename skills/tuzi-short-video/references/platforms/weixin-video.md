# 微信视频号 (WeChat Channels)

## Video Specs

| Parameter | Value |
|-----------|-------|
| Aspect Ratio | 9:16 (竖屏优先) / 16:9 |
| Resolution | 1080x1920 |
| Duration | 15-60s |
| Format | MP4 |
| Max File Size | 100MB |

## Content Tips

- 竖屏 9:16 优先，信息流展示更大
- 也支持 16:9 横屏
- 15-30s 短视频完播率高
- 内容偏向知识分享、生活记录
- 封面和标题影响点击率
